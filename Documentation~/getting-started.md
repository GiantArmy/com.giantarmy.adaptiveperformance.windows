# Getting started

The GiantArmy Adaptive Performance Windows provider is a data provider for the [Adaptive Performance](xref:um-adaptive-performance) package.

## Installation

1. [Set up Adaptive Performance](xref:um-adaptive-performance-initial-setup) in your project.
2. Open the **Windows Settings** tab and, in the **Providers** section, select **Windows Provider**.

## Log messages

To enable provider log messages in development builds on Windows devices:

1. Select **Edit** > **Project Settings** > **Adaptive Performance** > **Windows**.
2. In the **Development Settings** foldout, enable **Windows Provider Logging**.
