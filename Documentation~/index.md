# GiantArmy Adaptive Performance Windows provider

The GiantArmy Adaptive Performance Windows provider extends [Adaptive Performance](xref:um-adaptive-performance) to Windows devices.

This package includes:
* Windows provider runtime integration for Adaptive Performance.
* Windows provider settings and editor workflow.
* Windows package documentation and tests.
